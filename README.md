# Labas Learner Website

Simple public website and privacy policy for the Labas Learner Android test app.

This site is intended for GitHub Pages and Google Play Console app verification/privacy links.

## Pages

- `index.html` - app information page
- `privacy.html` - privacy policy for the Android app

## Publish With GitHub Pages

1. Create a public GitHub repository named `labas-learner`.
2. Upload the files in this folder to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Set source to **Deploy from a branch**.
5. Choose branch **main** and folder **/ root**.

The site should become available at:

```text
https://markypickard.github.io/labas-learner/
```

The privacy policy should become available at:

```text
https://markypickard.github.io/labas-learner/privacy.html
```
